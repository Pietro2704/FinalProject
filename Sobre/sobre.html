FAQ - (Frequently Asked Question)

O que é DOM? +
DOM é a sigla para Document Object Model, que representa a estrutura de uma página web em forma de árvore de elementos.
Para que serve? +
O DOM permite que você acesse e manipule elementos HTML via JavaScript, tornando a página dinâmica.
Como praticar? +
Pratique manipulando elementos da página: altere textos, cores, crie listas, formulários e interações com eventos.
